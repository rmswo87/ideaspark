## ⚠️ 발견된 문제점

### 1. 서버 구조 복잡성 (중요)
**문제**: 로컬 개발 시 두 개의 서버를 실행해야 함
- API 서버 (`npm run dev:api`) - 포트 3000
- 개발 서버 (`npm run dev`) - 포트 5173

**원인**: Reddit API CORS 문제로 인해 서버 사이드에서 호출해야 함

**해결 방안**:
1. **Vercel Edge Function 사용** (권장)
   - Vercel 배포 후 Edge Function으로 Reddit API 호출
   - 로컬 개발 시 `vercel dev` 사용
   - 단일 명령어로 모든 서버 실행

2. **Vite 프록시 사용** (임시)
   - Vite의 프록시 기능 활용
   - 단일 개발 서버로 통합

### 2. Vercel 배포 미완료 (중요)
**문제**: Vercel 배포가 안 되어 있어서:
- Reddit API Secret을 Vercel 환경변수에 설정할 수 없음
- 프로덕션 환경에서 테스트 불가
- Edge Function 사용 불가

**해결 방안**:
1. Vercel 프로젝트 생성 및 배포
2. 환경변수 설정 (Reddit API 키)
3. Edge Function 배포

### 3. 실제 데이터 수집 테스트 미완료
**문제**: 실제 Reddit API를 통한 데이터 수집이 테스트되지 않음

**해결 방안**:
1. Vercel 배포 완료 후 테스트
2. 또는 로컬 API 서버 실행 후 테스트
