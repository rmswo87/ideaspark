## 🎨 구체적 개선 사항

### 카드 디자인
- **현재**: 기본 Card 컴포넌트
- **개선**: 
  - 얇은 테두리 (border)
  - 부드러운 그림자 (shadow-sm hover:shadow-md)
  - 호버 시 약간의 상승 효과 (transform translateY)
  - 라운드 코너 (rounded-lg)

### 필터 UI
- **현재**: 기본 Select, Button
- **개선**:
  - 깔끔한 드롭다운 디자인
  - 활성 필터 배지 표시
  - 필터 그룹 시각적 구분

### 통계 카드
- **현재**: 텍스트 중심
- **개선**:
  - 아이콘 + 숫자 레이아웃
  - 색상 코딩
  - 호버 효과

### 버튼
- **현재**: 기본 Button 컴포넌트
- **개선**:
  - 부드러운 호버 효과
  - 활성 상태 명확히 표시
  - 로딩 상태 개선
