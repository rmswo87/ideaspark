## 📝 Vercel Cron Job 설정 가이드

### vercel.json에 cron 추가

```json
{
  "crons": [
    {
      "path": "/api/collect-dev-news",
      "schedule": "0 0 * * *"
    }
  ]
}
```

**스케줄 옵션**:
- `0 0 * * *`: 매일 자정 (Daily)
- `0 0 * * 0`: 매주 일요일 자정 (Weekly)
- `0 0 1 * *`: 매월 1일 자정 (Monthly)
