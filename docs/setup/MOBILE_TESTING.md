# 모바일 테스트 가이드

## 📱 모바일에서 로컬 개발 서버 접속하기

### 1. 네트워크 확인

PC와 모바일 기기가 **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다.

### 2. PC IP 주소 확인

#### Windows
```powershell
# PowerShell에서 실행
ipconfig

# 또는
ipconfig | findstr IPv4
```

#### Mac/Linux
```bash
ifconfig | grep "inet "
```

**예시 출력:**
- `192.168.0.100` (사설 IP - 같은 네트워크에서 사용)
- `124.57.33.108` (공인 IP - 인터넷을 통해 접속)

### 3. 개발 서버 실행

```bash
npm run dev
```

서버가 시작되면 다음과 같은 메시지가 표시됩니다:
```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.0.100:5173/
  ➜  Network: http://124.57.33.108:5173/
```

### 4. 모바일에서 접속

모바일 기기의 브라우저에서 다음 주소로 접속:

**같은 네트워크 (Wi-Fi)인 경우:**
```
http://192.168.0.100:5173
```
(실제 사설 IP 주소로 변경)

**공인 IP를 사용하는 경우:**
```
http://124.57.33.108:5173
```

### 5. 방화벽 설정 (필요한 경우)

#### Windows 방화벽
1. **Windows 보안** → **방화벽 및 네트워크 보호**
2. **고급 설정** 클릭
3. **인바운드 규칙** → **새 규칙**
4. **포트** 선택 → **다음**
5. **TCP** 선택, **특정 로컬 포트**: `5173` 입력
6. **연결 허용** 선택
7. **도메인, 개인, 공용** 모두 체크
8. 이름: "Vite Dev Server" 입력

#### PowerShell로 빠르게 설정 (관리자 권한)
```powershell
New-NetFirewallRule -DisplayName "Vite Dev Server" -Direction Inbound -LocalPort 5173 -Protocol TCP -Action Allow
```

### 6. 문제 해결

#### 모바일에서 접속이 안 되는 경우

1. **같은 네트워크 확인**
   - PC와 모바일이 같은 Wi-Fi에 연결되어 있는지 확인

2. **방화벽 확인**
   - Windows 방화벽이 5173 포트를 차단하고 있는지 확인
   - 바이러스 백신 소프트웨어의 방화벽도 확인

3. **IP 주소 확인**
   - `ipconfig`로 현재 IP 주소 확인
   - 공유기 재시작 후 IP가 변경되었을 수 있음

4. **포트 확인**
   - 다른 프로그램이 5173 포트를 사용하고 있는지 확인
   - Vite가 다른 포트를 사용 중일 수 있음 (콘솔 메시지 확인)

5. **Vite 설정 확인**
   - `vite.config.ts`에 `host: '0.0.0.0'` 설정이 있는지 확인

#### 공인 IP 사용 시 주의사항

- 공인 IP(`124.57.33.108`)는 인터넷을 통해 접속 가능하므로 보안에 주의
- 개발 중에만 사용하고, 완료 후에는 방화벽 규칙 제거 권장
- 프로덕션 배포 시에는 Vercel 등 안전한 호스팅 서비스 사용

### 7. 보안 팁

1. **개발 중에만 사용**
   - 로컬 개발 서버는 보안이 약하므로 개발 중에만 사용

2. **HTTPS 사용 (선택사항)**
   - 프로덕션 환경에서는 HTTPS 필수
   - 로컬 개발 시에는 HTTP로 충분

3. **방화벽 규칙 관리**
   - 개발 완료 후 방화벽 규칙 제거 또는 비활성화

---

**참고**: 
- Vite 개발 서버는 기본적으로 모든 네트워크 인터페이스에서 접속 가능하도록 설정되어 있습니다 (`host: '0.0.0.0'`)
- 같은 네트워크에 있으면 사설 IP 사용을 권장합니다 (더 빠르고 안전)
